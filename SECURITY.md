# Security Policy

## Sicherheitslücken melden

Bitte Sicherheitslücken **nicht** als öffentliches GitHub-Issue eröffnen.

Stattdessen vertraulich melden via GitHub *Security Advisory*: Tab „Security" → *Report a vulnerability*.

Bitte folgende Infos beifügen:

- Beschreibung der Schwachstelle
- Schritte zur Reproduktion
- Betroffene Version / Commit
- Mögliche Auswirkungen
- Vorschlag zur Behebung, falls vorhanden

Eingang wird innerhalb von 72 Stunden bestätigt. Nach Behebung kann der Reporter — wenn gewünscht — in den Release-Notes erwähnt werden.

## Unterstützte Versionen

Nur die aktuelle Hauptversion (`main`) sowie das letzte stabile Release erhalten Sicherheitsupdates.

## Datenschutz

MerchShop verarbeitet personenbezogene Daten (Name, E-Mail, Lieferadressen, Bestellhistorie). Betreiber sind für DSGVO-Konformität verantwortlich — siehe Design-Doc Abschnitt 10.
